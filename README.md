# 🃏 NPC Cards — Коллекционные карточки

**EN:** Collectible NPC cards for Foundry VTT. The GM creates character cards (art, quote, one of five visual templates) and grants them to players with a gacha-style reveal animation. Players build up a personal collection of every NPC they have met.

**Install:** Foundry VTT → Add-on Modules → Install Module →
```
https://github.com/BurnSIx-hub/npc-cards/releases/latest/download/module.json
```

---

## Что это (RU)

Система коллекционных карточек NPC: ГМ создаёт карточки персонажей и выдаёт их игрокам с эффектной анимацией в гача-стиле. У каждого игрока копится своя коллекция встреченных персонажей.

### Возможности

- 🎴 **Создание карточек** — имя, арт, описание/цитата
- ✨ **5 шаблонов оформления** — Тёмная Магическая, Тёмная Приключенческая, Серебряная Мистическая, Светлая Божественная, Картографическая
- 🎰 **Гача-выдача** — карточка появляется у игрока с анимацией открытия
- 👥 **Выдать одному или всем** — на выбор ГМ
- 📚 **Коллекция** — игрок в любой момент просматривает свои карточки

### Как пользоваться

1. ГМ: кнопка коллекции на панели → «Создать карточку»
2. Заполните имя, арт (URL или путь к файлу в Data), цитату, выберите шаблон
3. «Выдать игроку» / «Выдать всем» — у игроков проиграется анимация получения
4. Игроки открывают «Мою коллекцию» той же кнопкой на панели

### Установка вручную

Скачайте `module.zip` из [последнего релиза](https://github.com/BurnSIx-hub/npc-cards/releases/latest) и распакуйте в `FoundryVTT/Data/modules/npc-cards/`.

Требования: Foundry VTT v13–v14.
